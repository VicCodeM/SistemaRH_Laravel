Manual de Usuario

Bienvenido al Sistema de Recursos Humanos (SistemaRH). Aqui encontraras todo lo necesario para usar el sistema segun tu rol.

SistemaRH v1.0

Acceso al Sistema

Para entrar a SistemaRH solo necesitas un navegador web y tus credenciales.

1
Abrir la pagina

Escribe la direccion del sistema en tu navegador. Si lo usas en tu computadora sera algo como:
http://localhost:8000

2
Iniciar sesion

Haz clic en Iniciar sesion y escribe tu correo y contrasena.

3
Entrar a tu panel

El sistema te lleva automaticamente al panel que corresponde a tu rol: Admin, Empresa, Candidato o Interno.

💡
Consejo

Todas las contrasenas de prueba son password. En produccion cada usuario tiene su propia contrasena segura.

Administrador

El Administrador es el encargado principal del sistema. Revisa, aprueba y gestiona todo lo que hacen las empresas, los candidatos y el personal interno.

  • 📊
    Ver estadisticasCuanto empresas, candidatos, vacantes y tareas activas hay en el sistema.
  • 🏢
    Aprobar o rechazar empresasCuando una empresa nueva se registra, el admin debe revisar sus datos y darle acceso.
  • 👤
    Aprobar o rechazar candidatosRevisa la solicitud de empleo de cada candidato y decide si puede buscar trabajo.
  • 📋
    Activar vacantesCuando una empresa pide una vacante (servicio de reclutamiento), el admin la revisa y la activa para que los candidatos la vean.
  • 🔍
    Buscar candidatos compatiblesEl sistema sugiere candidatos que se ajustan a una vacante segun sus habilidades, ciudad y experiencia.
  • 📌
    Crear y asignar tareasCuando una empresa o candidato pide un servicio, el admin crea una tarea y la asigna a un empleado interno.
  • 👥
    Gestionar personal internoRegistra nuevos empleados internos y define que servicios pueden realizar.
  • 📚
    Configurar catalogosCambia las opciones de los menus desplegables sin necesidad de tocar el sistema.
  • 📈
    Generar reportesDescarga listados de empresas, candidatos, vacantes y tareas.
💡
Importante

Nada se activa en el sistema sin la aprobacion del Administrador. Es el pivote central de todo el flujo.

Empresa

Las empresas son los clientes que piden servicios de Recursos Humanos: reclutamiento (vacantes), capacitaciones, coaching y mas. No gestionan el proceso directamente; lo solicitan y el equipo de RH (Admin) se encarga.

1. Registro: La empresa llena un formulario con sus datos fiscales (RFC, direccion, telefono) y espera a que el Admin la apruebe.

2. Pedir una vacante: Una vez aprobada, solicita el servicio de reclutamiento para una vacante. El Admin la revisa y la activa para que los candidatos la vean.

3. Ver el avance: Consulta el estado de sus solicitudes y vacantes, y quien fue asignado o seleccionado. El proceso de seleccion lo gestiona el equipo de RH (Admin); la empresa solo lo consulta.

4. Solicitar otros servicios: Puede pedir capacitacion, coaching, evaluacion o consultoria para su personal.

5. Chatear con el Admin: Tiene un chat directo para comunicarse con el Administrador.

⚠️
Restriccion

Si la empresa esta pendiente o rechazada, no puede pedir servicios ni usar el sistema.

Candidato

Los candidatos son personas que buscan empleo. Llenan una solicitud de empleo detallada y se postulan a las vacantes disponibles.

1. Crear cuenta: Se registra con su correo y contrasena.

2. Llenar solicitud de empleo: Completa 5 secciones: datos personales, contacto, estudios, historial laboral y extras.

3. Enviar solicitud: El Admin revisa y aprueba. Si no esta aprobada, no puede postularse.

4. Buscar vacantes: Ve las vacantes activas que el sistema le recomienda segun su perfil.

5. Postularse: Selecciona una vacante y presiona "Postularme". El equipo de RH (Admin) revisa tu perfil.

6. Seguimiento: Ve el estado de sus postulaciones: en revision, entrevista, seleccionado o rechazado.

⚠️
Restriccion

Un candidato solo puede postularse si su solicitud esta aprobada. Mientras este en borrador o pendiente, no puede aplicar a vacantes.

Personal Interno

El personal interno son los empleados operativos que ejecutan los servicios solicitados. Reciben tareas del Administrador y las completan.

1. Recibir tarea: El Admin le asigna una tarea (por ejemplo: capacitar al personal de una empresa).

2. Tomar la tarea: En su panel ve la tarea y presiona "Tomar" para empezar a trabajar.

3. Ejecutar: Realiza el servicio: entrevistas, capacitaciones, evaluaciones, reclutamiento, etc.

4. Completar o Cancelar: Cuando termina, escribe un resumen y marca la tarea como completada. Si no puede hacerla, la cancela con un motivo.

⚠️
Restricciones

El interno solo ve sus propias tareas. No puede crear tareas, ni ver informacion de empresas o candidatos. Todo lo recibe del Administrador.

Chat Interno

El chat es una mensajeria directa para comunicarse dentro del sistema. No es como WhatsApp, tiene reglas especificas.

  • Solo chat uno a uno: No hay grupos. Cada conversacion es entre dos personas.
  • El Admin puede chatear con todos: Puede iniciar conversacion con cualquier empresa, candidato o interno.
  • Los demas solo chatean con el Admin: Una empresa no puede hablar con un candidato. Un interno no puede hablar con una empresa. Todo pasa por el Administrador.
  • Conversacion automatica: Si un usuario que no es Admin abre el chat, el sistema crea solo la conversacion con el Admin.
⚠️
Importante

El chat es privado entre cada usuario y el Admin. No hay canales publicos ni grupos.

Resumen por rol

🛡️ Admin: Aprueba todo. Revisa empresas, candidatos, vacantes y servicios. Crea y asigna tareas.

🏢 Empresa: Pide servicios (reclutamiento, capacitacion...) y consulta el avance de sus solicitudes. No publica ni gestiona; eso lo hace el equipo de RH.

👤 Candidato: Llena su solicitud, espera aprobacion y se postula a vacantes.

🔧 Interno: Recibe tareas, las ejecuta y las cierra con un resumen.
